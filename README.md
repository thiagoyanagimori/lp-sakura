Yeston Sakura RTX 5080 — Interactive Landing Page

An experimental cinematic product landing page inspired by Apple's product storytelling and modern hardware presentation websites.

This project was created to explore:

cinematic scroll animations
premium minimalist UI
immersive product storytelling
GSAP-powered motion design
exploded-view GPU presentation
high-end frontend experiences

The landing page showcases the Yeston Sakura RTX 5080 through smooth transitions, atmospheric visuals, and interactive engineering sections.
